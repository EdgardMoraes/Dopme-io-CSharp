# Dopme-io C# — Mentoria em Desenvolvimento .NET

Repositório de aprendizagem prática em C# e .NET Framework, desenvolvido durante mentoria
formal com a [dopme.io](https://dopme.io).

## Sobre este repositório

Este repositório documenta a minha evolução como desenvolvedor .NET, com exercícios,
projetos práticos e exemplos trabalhados ao longo da mentoria. O objetivo é consolidar
boas práticas de desenvolvimento em C# aplicadas a contextos reais.

## Tecnologias

- **Linguagem:** C#
- **Framework:** .NET Framework
- **Ferramentas:** Visual Studio, Git

## Conteúdos abordados

- [x] Fundamentos de C# e tipos de dados
- [x] Programação Orientada a Objetos (OOP) — classes, herança, encapsulamento, polimorfismo
- [x] Coleções e LINQ
- [x] Tratamento de exceções
- [ ] Acesso a dados com Entity Framework
- [ ] Padrões de projeto (Design Patterns)
- [ ] Boas práticas e clean code

> ✅ Itens marcados já foram trabalhados. O repositório é atualizado continuamente.
